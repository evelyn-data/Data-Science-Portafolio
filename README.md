### Featured Project

#### 📊 [Sales Analysis & Interactive Dashboard — TechCore](./M3_TechCore-Sales-Analysis/)

> End-to-end data project for TechCore, a retail chain specialized in computers and tech accessories. Built a complete pipeline from raw billing data to an interactive Power BI dashboard.

**Tech stack:** Power BI Desktop · Power Query (M) · DAX · Python (Pandas, NumPy) · Excel

**Key deliverables:**
1. **Data Cleaning & Transformation** — Standardized and cleaned raw billing data using Power Query. Handled nulls, duplicates, data types, and categorical inconsistencies.
2. **Relational Modeling** — Designed and implemented a normalized relational model (Invoices, Products, Clients, Branches, Sellers) with validated referential integrity.
3. **Interactive Dashboard** — Built a Power BI dashboard with KPIs (Total Sales, Average Ticket, Sales by City, Market Share %) using DAX measures for dynamic filtering.

### Proyecto Destacado

#### 📊 [Análisis de Ventas y Dashboard Interactivo — TechCore](./M3_TechCore-Sales-Analysis/)

> Proyecto end-to-end para TechCore, una cadena minorista especializada en computadores y accesorios tecnológicos. Se construyó un pipeline completo desde los datos crudos de facturación hasta un dashboard interactivo en Power BI.

**Tech stack:** Power BI Desktop · Power Query (M) · DAX · Python (Pandas, NumPy) · Excel

**Entregables principales:**
1. **Limpieza y Transformación de Datos** — Estandarización y limpieza de la base de datos cruda de facturación con Power Query. Tratamiento de nulos, duplicados, tipos de datos e inconsistencias categóricas.
2. **Modelo Relacional** — Diseño e implementación de un modelo relacional normalizado (Facturas, Productos, Clientes, Sucursales, Vendedores) con integridad referencial validada.
3. **Dashboard Interactivo** — Dashboard en Power BI con KPIs (Ventas Totales, Ticket Promedio, Ventas por Ciudad, % de Participación) mediante medidas DAX con filtros dinámicos.
